## 📖 About the Project

Buying or selling a car can be tricky without knowing its fair value. This project uses machine learning techniques to build a predictive model that estimates the price of a car based on historical data.

---

## ✨ Features

* Data cleaning and preprocessing
* Exploratory data analysis (EDA) with visualizations
* Model training and evaluation (Linear Regression, Random Forest, XGBoost)
* Streamlit web app for interactive predictions

## ⚙️ Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/username/car-price-prediction.git
cd car-price-prediction
pip install -r requirements.txt
```

---

## 🚀 Usage

Run the Streamlit app:

```bash
streamlit run app.py
```

Then open the link in your browser: `http://localhost:8501/`

---

## 📊 Results

* Random Forest achieved the best performance with an **R² score of 0.87**.
* Feature importance showed that *mileage* and *engine size* are the strongest predictors.
